Задание на вёрстку письма
Вёрстка письма по макету. Требования:

Современные браузеры
Максимальное соответствие макету
Использование CSS не допускается
Управление отображением элементов - при помощи атрибутов
Максимальная ширина основного контента 560px
Каждый логический блок - отдельная, вложенная, таблица - это позволит отключать, либо менять местами, блоки
Для управления гарнитурой и цветом текста используйте элемент с атрибутами
Для управления размерами текста используйте семантические элементы - параграфы и заголовки
В качестве гарнитуры шрифта указывайте наименование семейства шрифта
Блоки с нестандартными шрифтами используйте в качестве изображения
Скругленные края и кнопки - также вставляйте изображениями
Сплошные области таблицы (без скруглений) - должны быть залиты цветом
Используйте ряды и ячейки пустышки для установки пространства и отступов
P.S Валидацию на 100% выбивать не нужно!
